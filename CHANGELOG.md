# Changelog

## 2026-02-24 — JSON-тесты и inline-вопросы в статьях

### Добавлено
- **JSON-формат тестов** (`src/art/test/ai-*.json`) — вопросы хранятся отдельно от MD
- **Inline-вопросы** — маркер `<!-- q:ID -->` в MD вставляет вопрос между абзацами
- **Тест после статьи** — вопросы без маркеров автоматически попадают в `<Test>` с таймером
- **mapTestData.js** — утилита конвертации JSON → Quest-props с автохэшированием ответов
- **6 JSON-файлов тестов** — по 7 вопросов на статью (1 inline + 6 в тесте), 9 типов вопросов
- Поддержка коротких типов: `choice`, `multi-choice`, `fill`, `text`, `code`, `order`, `pairs`, `spot`, `blocks`, `true-false`

### Изменено
- **parseArticle.js** — заголовок из `#`, извлечение маркеров `<!-- q:ID -->`, обратная совместимость со старым форматом
- **Article.jsx** — рендеринг inline-вопросов между сегментами ReactMarkdown, тест после статьи
- **MD-файлы** — убраны «Статья N», «Шокирующий заголовок:», «Текст статьи:», секции «Опросник:»/«Тест:»

### Формат JSON-вопроса
```json
{ "id": "1", "type": "choice", "question": "...", "items": [...], "answer": 1, "hint": "...", "points": 2 }
```
ID вопросов — простые числа (1, 2, 3...), уникальность в localStorage обеспечивается префиксом articleId.
